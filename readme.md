# My Multipage Website

## Overview
This is a responsive multipage website built with **HTML5, CSS3, and JavaScript**. It includes interactive elements such as buttons, hover effects, and a contact form with validation.

## Pages
- `index.html` - Home page with hero and features sections.
- `about.html` - About page describing the website/project.
- `contact.html` - Contact page with interactive form validation.

## Features
- Mobile responsive design
- CSS hover animations and transitions
- JavaScript interactivity (alerts, form validation, mobile menu toggle)
- Consistent header and footer across all pages

## Deployment
This website can be deployed on **GitHub Pages, Netlify, or Vercel**. All links, scripts, and styles are relative paths, ready for deployment.
